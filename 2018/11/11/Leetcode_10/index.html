416 Partition Equal Subset Sum

评价

中等

题目

Given a non-empty array containing only positive integers, find if the array can be partitioned into two subsets such that the sum of elements in both subsets is equal.

Note:
Each of the array element will not exceed 100.
The array size will not exceed 200.

题目理解

这道题又是看起来很简单，做起来有一点点难（好吧，我还是觉得挺难的，因为刚学动态规划…），给一个数组，然后我们需要将数组分成和相等的两部分。

一开始看觉得怎么会划分到动态规划那里，这种不应该是DFS的题目吗，后来偷偷看了一眼别人写的博客，发现居然是当成0/1背包问题去做，即一个物品只有两个选项，放进背包里，还是不放进背包里。看到这里，我良心发现，要变强，怎么能偷看答案呢？

我就继续开始想，首先想的是很简单，因为要分成两部分，是否可以假设两个背包，把数字分别放在两个背包里面，但是这样的话会很麻烦，我也感觉不会，老师上课只教过一个背包的，我们如何将它转变成一个背包呢？我突然想到，可以利用一个大小等于数组总和一半的背包，不就可以了吗？只要找到一组数，装满它，问题就完美解决了。

代码

首先需要判断数组和是否为偶数，否则肯定无法分成大小相等的两部分，接着假设一个背包的大小为数组总和的一半，对于数组中任意一个数字，我们都可以更新一次背包可达到的容量，比如已知背包的容量可以达到k，如果放一个物品大小为x，那么背包可以容纳k + x的物品，如果背包可以装满，则返回true。

额外学到的东西

  1. c++中对一个数组中所有元素求和，可以numeric库中的accumulate(nums.begin(), nums.end(), 0)，可以节省些打代码的时间。

  2. 想要判断一个数字是奇数还是偶数，可以利用number & 1 != 0去判断。
