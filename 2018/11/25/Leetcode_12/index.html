647 Palindromic Substrings

评价

中等

题目

Given a string, your task is to count how many palindromic substrings in this string.

The substrings with different start indexes or end indexes are counted as different substrings even they consist of same characters.

Note:

The input string length won’t exceed 1000.

题目理解

输入字符串，输出其中是回文的子字符串的数量

版本 1

首先我写了一个复杂度为 O(n^3) 的方法，即对每一长度的子字符串都进行判断，从长度为 1 的子字符串开始，直到整个字符串为止，虽然很慢，但还是 AC 了。

版本 2

后来我想到了这种方法，没有利用到动态规划，但还是比之前那个快，是 O(n^2) 的方法，对字符串中的每一个字符，算出以这个字符中心对称的子字符串（奇数长度）的回文数量，再算以这个字符和相邻右边的一个字符中心对称的子字符串（偶数长度）的回文数量，最后相加，则得到答案。

版本 3

最后我还是没有想到动态规划的方法，所以我看了一下 discussion 里面的人，看到大家是这样做的，先生成一个 n x n 维的布尔数组 dp，定义 dp[i][j] 为字符串中从 i 到 j 的子字符串是否为回文，i 从数组末尾开始减小，j 从 i 开始增大，如果 i 和 j 间隔不超过 3 时，如果 s[i] 和 s[j] 相等，则为回文，如果 i 和 j 间隔超过 3 时，则不仅需要 s[i] 和 s[j] 相等，还需要他们之间的子字符串为回文才行。

额外学到的东西

string 可以使用 string substr (size_t pos = 0, size_t len = npos) const 函数进行产生子字符串

注意：

  1. 第二个参数是长度

  2. 返回值是一个 const 的字符串
