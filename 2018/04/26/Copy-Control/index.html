Copy, Assign, and Destroy

The Copy Constructor

Definition :
The first parameter is a reference to the class type and any additional parameters have default values.

ps. The first parameter is almost always a reference to const.

Note :
The compile can bypass the copy constructor.

Synthesized Copy Constructor

Usage :
Members of class type are copied by the copy constructor for that class while members of build-in type are copied directly.

Note :

  1. Even if we define other constructors, a copy constructor is synthesized when we don’t define copy constructors.
  2. Although we can’t directly copy an array, synthesized copy constructor copies members of array type by coping each element (a deep copy).

Example :

Conclusion :

  1. Define variables using an =
  2. Pass an object as an argument to a parameter of nonreference type
  3. Return an object from a function that has a nonreference return type
  4. Brace initialize the elements in an array or the members of an aggregate class

Note :

  1. Copy initialization ordinarily uses the copy constructor. However copy initialization doesn’t equal to copy constructor because sometimes it may use move constructor.
  2. In some container libraries, insert or push refers to copy initialize and emplace refers to direct initialize.

Copy-Assignment Operator

定义 :
将一个对象用等号赋值给另外一个对象时所发生的动作。

区别 :
Copy-Assignment operator和Copy constructor的区别在于，前者在赋值时调用，后者在初始化时调用。

使用 :
重载等号 = 操作符的功能

例子:

等号=的left-hand operand绑定在this上，right-hand operand 绑定在参数上

ps. 之所以要返回一个left-hand operand的引用，是为了和build-in type保持一致，可以实现a = b = 1的这种功能
