Python包中init.py作用

在创建包的过程中，IDE都会在包根目录下创建一个init.py文件，该py文件默认是空的，作用如下：

  1. package的标识，如果没有该文件，该目录就不会认为是package；
  2. 定义package中的all变量，用来模糊导入。

注意：因为init.py仍然是个py文件，所以还可以用来写python模块，但是不建议这么写，尽量保证init.py足够轻。
