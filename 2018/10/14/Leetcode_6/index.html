513 Find Bottom Left Tree Value

评价

中等

题目

Given a binary tree, find the leftmost value in the last row of the tree.

题目理解

需要找到最下一层中最左边的数字。

解析

利用BFS算法从左到右找到最下面一层，除了BFS所需的队列外，存储两个临时值，一个代表该层剩余的元素个数，一个代表值，当到下一层时则替代掉临时值继续处理。
