832 Flipping an Image

评价

简单

题目

Given a binary matrix A, we want to flip the image horizontally, then invert it, and return the resulting image.

To flip an image horizontally means that each row of the image is reversed. For example, flipping [1, 1, 0] horizontally results in [0, 1, 1].

To invert an image means that each 0 is replaced by 1, and each 1 is replaced by 0. For example, inverting [0, 1, 1] results in [1, 0, 0].

题目理解

将布尔矩阵先左右反转，再取反

注意点

  1. 每行先反转再取反的时，可以先将首尾交换，然后交换的时候顺便取反，这样可以省一步

  2. 因为是0和1的交换，所以可以直接用 1 - x，或者 x ^ 1

  3. 当使用注意点1来首尾交换并取反时，不能直接用长度除以2作为范围，否则当长度为奇数的时候，最中间那个数会被忽略，所以要使用长度加1除以2才行

评价

题目

Given a triangle, find the minimum path sum from top to bottom. Each step you may move to adjacent numbers on the row below.

For example, given the following triangle

The minimum path sum from top to bottom is 11 (i.e., 2 + 3 + 5 + 1 = 11).

Note:

Bonus point if you are able to do this using only O(n) extra space, where n is the total number of rows in the triangle.

题目理解

在一个三角形中，从最上面的元素开始，到最下面一层结束，找出权重最小的路径

注意点

从底到顶：可以利用一个长度为n的数组，然后记录从底向上的路径向量，该数组初始化为三角形的最底层，接着每轮循环，记录向上一层路径的最小值，直到三角形的顶部
