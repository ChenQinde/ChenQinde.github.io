基于 Ethereum 实现货币系统

发表于 2019-06-26 | 分类于 课程博客

简介

  • Ethereum 是一个开源的有智能合约功能的公共区块链平台，提供去中心化的 Ethereum Virtual Machine 来处理点对点合约。

  • ERC20 是一种以太坊上的代币标准，用以评判该代币是否满足一定的安全性和可用性。

  • Solidity 是一门面向合约的、为实现智能合约而创建的高级编程语言。它属于静态类型语言，支持继承、库和复杂的用户定义类型等特性。

依赖环境

Solidity 0.5.0 以上

以上程序都是用来处理 uint256 类型数据的四则运算，并返回 uint256 类型数据，所以需要保证结果为非负数，且大于该类型能存储的数据范围才行。

Coin.sol

实现 ERC20

  1. ERC20 标准 API，实现了以下四个功能
    • tranfer：自己本地转账
    • approve：批准别人使用自己的钱（可多次使用）
    • transferFrom：与approve结合使用
    • allowance：返回批准使用钱的余额
  1. Coin 货币系统的实现，有以下的设计：
    • 总供量：1000000000
    • 名字：Coin
    • 符号：C
    • 精确小数位：0（整数）
